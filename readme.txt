Gust is a port of Ghost admin panel for WordPress.

== Description ==

At the begining, Ghost was supposed to be a next-generation admin panel for WordPress, but in the process it became a new blogging platform, built on Node.js. This plugin is an attempt to bring it back to the WordPress ecosystem. 

Current features:

*   Posts/Pages Markdown editor
*   Media upload integration

== Installation ==

1. Upload `gust` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Make sure pretty permalinks are enabled
1. Go to yourdomain.com/gust
